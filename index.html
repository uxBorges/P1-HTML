Tags de Texto

Retornar

As tags de texto HTML são elementos que definem a estrutura e o conteúdo de um documento web. Elas são escritas entre sinais de menor e maior (< e >) e formam pares que delimitam o início e o fim de um elemento. Por exemplo, a tag <p> indica o início de um parágrafo e a tag </p> indica o fim.

As tags têm atributos que especificam propriedades adicionais do elemento, como o seu estilo, classe ou identificador.
Por exemplo, a tag <img> tem o atributo src que indica a fonte da imagem a ser exibida. As tags HTML podem ser aninhadas umas dentro das outras para criar elementos mais complexos e organizados.

Por exemplo, a tag <div> pode conter várias tags <p> para formar uma seção de texto. Existem muitas tags HTML diferentes que servem para diferentes propósitos, como criar cabeçalhos, listas, tabelas, formulários, links, etc. Para saber mais sobre as tags HTML, consulte a referência oficial do W3C ou algum tutorial online.

Tags de Mídia

Retornar

As tags de mídia em HTML são elementos que permitem incorporar arquivos de áudio, vídeo e imagens em uma página web. O HTML5 introduziu cinco tags de mídia principais:

<audio>, <video>, <source>, <embed> e <track>

.

A tag <audio> é usada para embutir arquivos de som em uma página web. Ela pode receber atributos como src, controls, autoplay, loop e muted. Por exemplo: <audio src=\"musica.mp3\" controls autoplay loop></audio>

A tag <video> é usada para incorporar arquivos de vídeo em uma página web. Ela pode receber atributos como src, controls, autoplay, loop, muted, width, height e poster. Por exemplo: <video src=\"filme.mp4\" controls width=\"800\" height=\"600\" poster=\"capa.png\"></video>

A tag <source> é usada para anexar arquivos multimídia como áudio, vídeo e imagens. Ela deve ser usada dentro das tags <audio> ou <video> para especificar fontes alternativas para os arquivos de mídia. Ela pode receber atributos como src e type.

Por exemplo: <video controls> <source src=\"filme.mp4\" type=\"video/mp4\"> <source src=\"filme.webm\" type=\"video/webm\"> </video> A tag <embed> é usada para incorporar qualquer tipo de conteúdo externo em uma página web, como arquivos Flash, Java applets, PDFs, etc.

Ela pode receber atributos como src, width, height e type. Por exemplo: <embed src=\"jogo.swf\" width=\"800\" height=\"600\" type=\"application/x-shockwave-flash\"></embed>

A tag <track> é usada para adicionar legendas ou descrições a um arquivo de áudio ou vídeo. Ela deve ser usada dentro das tags <audio> ou <video> e pode receber atributos como src, kind, srclang e label.

Por exemplo: <video src=\"filme.mp4\" controls> <track src=\"legenda.vtt\" kind=\"subtitles\" srclang=\"pt\" label=\"Português\"> </video>

Tags de Links

Retornar

As tags de link em HTML são usadas para especificar as relações entre o documento atual e um recurso externo, como uma folha de estilo, um ícone ou uma pré-carga.

A tag <link> é um elemento vazio que deve ser colocado dentro do elemento <head> do documento.

A tag <link> tem vários atributos, mas os mais comuns são href, rel e type. O atributo href define o endereço do recurso vinculado, o atributo rel define o tipo de relação entre o documento e o recurso, e o atributo type define o tipo de mídia do recurso.

Por exemplo, para vincular uma folha de estilo externa ao documento, podemos usar a seguinte tag:

<link href="estilo.css" rel="stylesheet" type="text/css">


Neste caso, o atributo href aponta para o arquivo estilo.css, o atributo rel indica que o recurso é uma folha de estilo, e o atributo type indica que o recurso é um arquivo de texto com formato CSS.

Existem outros valores possíveis para os atributos rel e type, dependendo do tipo de recurso que se quer vincular.

As tags de link em HTML são úteis para melhorar a apresentação, a funcionalidade e a acessibilidade dos documentos web.
